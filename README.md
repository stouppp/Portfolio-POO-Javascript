# ☕ Portfolio de Programmation Orientée Objet - Java

Bienvenue sur mon portfolio dédié à la Programmation Orientée Objet (POO). Ce dépôt regroupe 4 projets distincts développés en Java, illustrant ma progression et ma maîtrise de différents concepts logiciels : de la modélisation de base à l'algorithmique avancée, en passant par les interfaces graphiques.

## 📂 Projets inclus

* 🏎️ **Projet 1 : Simulation d'Écurie Automobile**
  Modélisation complète d'une écurie (véhicules, staff, assignations) mettant en pratique les piliers de la POO (Héritage, Polymorphisme, Encapsulation).
* 🎨 **Projet 2 : Mini-Paint interactif**
  Application de dessin vectoriel utilisant Java Swing et AWT, structurée autour de l'architecture Modèle-Vue-Contrôleur (MVC) et de la gestion événementielle.
* 🚚 **Projet 3 : Optimisation de Tournées (VRP)**
  Résolution algorithmique d'un problème de logistique (Vehicle Routing Problem) avec une utilisation avancée des Collections Java et la manipulation de flux de fichiers (I/O).
* ⏱️ **Projet 4 : Ordonnancement de Tâches**
  Simulateur d'allocation de tâches sur machines parallèles. Ce projet met en œuvre le **Design Pattern Strategy** pour comparer facilement différentes heuristiques d'optimisation.

> **💡 Note :** Vous trouverez les détails techniques, l'architecture et les commandes d'exécution dans le fichier `README.md` situé à l'intérieur de chaque dossier de projet.
